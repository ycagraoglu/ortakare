# Ortakare Backend

Bu klasör ASP.NET Core 10 LTS tabanlı Ortakare API uygulamasını içerecektir.

Kesin mimari kararlar için:

- [`ADR-001 — Backend Architecture and Coding Principles`](../docs/architecture/ADR-001-backend-architecture.md)

Planlanan solution yapısı:

```text
backend
├── Ortakare.sln
├── src
│   ├── Ortakare.Api
│   ├── Ortakare.Domain
│   └── Ortakare.Infrastructure
└── tests
    ├── Ortakare.UnitTests
    └── Ortakare.IntegrationTests
```
