# Ortakare Frontend

Bu klasör React, TypeScript ve Vite tabanlı Ortakare PWA uygulamasını içerecektir.

Uygulama iki ana deneyim sunacaktır:

- Etkinlik sahibinin giriş yaptığı yönetim alanı.
- Misafirlerin QR kod ile açtığı public katılım ve fotoğraf yükleme alanı.

Planlanan temel teknoloji yığını:

- React
- TypeScript
- Vite
- PWA
- TanStack Query
- shadcn/ui
